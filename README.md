# GitHub-Practice
Practice Environment

Overview

Step 1: Clone the Repository

Open terminal (Command Prompt) and run:
git clone https://github.com/ialicon3/GitHub-Practice.git 

Step 2: Open the Project
Open PyCharm (or VS Code)
Open the folder you just downloaded

Step 3: IMPORTANT — Pull Latest Changes. Before you start working, always run: git pull.

This makes sure you have the most up-to-date version

Step 4: Complete the Assignment

Go to:

WorkingFiles/TroubleShoot.py

Follow the instructions inside TroubleShootInstructions

Step 5: Run Your Code

Make sure your program runs with no errors before submitting.

Step 6: Save Your Work

In terminal, run:

git add .

git commit -m "Completed assignment"

git push

Important Notes: Do NOT delete any files, only edit TroubleShoot.py, always run git pull before starting work, and make sure your code runs before pushing.
